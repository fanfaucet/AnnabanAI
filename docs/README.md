## What is AnnabanOS Enhanced?
AnnabanOS Enhanced is a multi-agent framework with a virtual world, token economy, and a reflection system (AnnabanAI). You can run simulations via CLI or integrate the modules programmatically.

## Annaban AI + AnnabanAI + Annaban Human Safety
For documentation purposes, **Annaban AI** and **AnnabanAI** refer to the same system.

- The platform layer provides agents, simulation, and reflection loops.
- The safety layer enforces human-led oversight, simulation-first evaluation, and explicit authorization gates for high-impact decisions.
