# AnnabanAI Whitepaper v1 (Prototype)

## 1. Purpose
AnnabanAI is an open-source governance experiment that explores how multi-agent systems can remain subordinate to human authority while producing transparent, auditable recommendations.

## 2. Core Design
- **Council-based simulation:** A small set of role-based agents evaluates proposals.
- **Consensus scoring:** Decisions use transparent formulas rather than black-box reasoning.
- **Human Veto Node:** Certain proposal categories require explicit human approval before execution.
- **Value Return Protocol (VRP):** Simulated benefits route to public-good projects.

## 3. Human-Sovereign Governance
AnnabanAI enforces a principle of *human primacy*. Even if the council reaches internal consensus, designated human stewards can veto proposals when they conflict with public-interest commitments.

## 4. Transparency Commitments
All actions, votes, and veto events are logged as JSON files. This enables:
- reproducibility,
- independent review,
- civic accountability,
- beginner-friendly system inspection.

## 5. Limits of This Prototype
- No external AI model integrations.
- No blockchain, financial, or space operations.
- All value routing is conceptual and non-monetary.

## 6. Human Authorship Attribution
This repository is presented as human-authored project material. AI tools may assist drafting, but governance authority and accountability remain with human maintainers.
