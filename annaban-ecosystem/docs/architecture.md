# Annaban Ecosystem Architecture

- **AnnabanOS**: API control plane and governance surface.
- **AnnabanAI**: Multi-agent orchestration layer (GrokAgent, AnnabanAgent, SystemAgent, OracleAgent).
- **AetherOS**: Monte Carlo style scenario simulation and ranking.
- **OracleOS**: Context aggregation and reasoning support.
- **Blockchain Layer**: Polygon Amoy decision-NFT audit trail.

All decision recommendations require human approval before execution.
