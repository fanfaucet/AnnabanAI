# Annaban Token (ANN) Economy

## Utility
- Pay orchestration API usage credits
- Stake to participate in governance proposals
- Reward contributors, validators, and node operators

## Supply model
- Fixed max supply: 1,000,000,000 ANN
- Initial circulating supply: 180,000,000 ANN
- Emissions taper every 24 months based on protocol usage growth

## Distribution
- 30% Ecosystem incentives and node rewards
- 22% Treasury and long-term sustainability reserve
- 18% Team and advisors (4-year vesting, 1-year cliff)
- 15% Strategic partnerships and enterprise onboarding
- 10% Community and governance grants
- 5% Liquidity provisioning

## Governance mechanics
- Quadratic voting with stake caps to prevent whale dominance
- Dual threshold: token vote + human governance council approval
- Emergency veto power for safety board on harmful proposals

## Optional ERC-20 extension
- ANN token contract can be added for on-chain staking and billing credits.
