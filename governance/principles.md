# AnnabanAI Governance Principles

1. **Human Sovereignty First**  
   Human judgment overrides autonomous recommendations.

2. **Transparency by Default**  
   Decisions, votes, and exceptions must be visible in audit-friendly logs.

3. **Public-Benefit Orientation**  
   Simulated value flows through a Value Return Protocol toward social-good initiatives.

4. **Contestability and Review**  
   Every decision can be inspected, challenged, and revised by human participants.

5. **Open Participation**  
   Community contributors can propose improvements under ethical guardrails.
